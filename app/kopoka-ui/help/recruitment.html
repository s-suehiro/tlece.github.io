Manpower planning shows the result of workload analysis for each job title, with comparison with existing number of employees on each job title, and let operator use them as the basis of recruitment.

Below is the description of each item in this screen:

Search box                   
                		Type any keyword and only the job title containing the keyword will be shown
“Go to Recruitment Planning” button                   
                		When clicked brings user to “Recruitment Planning” screen                   
               
Job titles list                   
                		A list of job titles with man power planning data                   
               
“Job Title” field                   
                		Job title whose manpower planning is described                   
               
“Workload Analysis” field                   
                		The result of workload calculation for the job title. If workload analysis have not been calculated, there will be “not defined” mark                   
               
“Existing Employee” field                   
                		The number of existing employees for the job title                   
               
“Manpower Planning” field                   
                		The number of persons needed now for the position, either from succession, or internal/external recruitment. Therefore, this number is the total of the next 3 fields below, i.e., “Succession”, “Internal” and “External”                   
               
“Succession” field                   
                		The number of persons planned to fill the position through succession.                   
               
“Internal” field                   
                		The number of persons planned to fill the position through internal recruitment.                   
               
“External” field                   
                		The number of persons planned to fill the position through external recruitment.                   
               
“Action” field                   
                		Contains “Edit” button, for editing the no. of succession, internal and external.
